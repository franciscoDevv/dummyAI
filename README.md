## Usage
Clone this repository:

```
git clone https://github.com/greedyboyy/dummyAI
```
Go in!
```
cd dummyAI
```
Install [cohere](https://cohere.com) with pip
```
pip install cohere
```
Run it
```
python main.py
```
Enjoy!
